# 指令训练平台项目交付清单

## 项目概述

本项目是一个基于 Rasa 3.6.21 的智能指令训练平台，支持中文语义理解、模型训练和预测服务。项目采用现代化的技术栈，包括 React 前端、FastAPI 后端和 Rasa AI 引擎。
